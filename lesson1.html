Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using various elements and tags.

Basic HTML Structure:

<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

Key Concepts

  • HTML documents are made up of elements
  • Elements are defined by tags
  • Tags usually come in pairs (opening and closing tags)
  • Some elements are self-closing

Basic HTML

Mulai dengan dasar HTML. Pelajari struktur, tag, dan elemen dasar untuk membangun halaman web sederhana.

Intermediate HTML

Pelajari fitur HTML lanjutan seperti formulir kompleks, semantic HTML, dan teknik media embedding.

Advanced HTML

Dalami topik advanced seperti Web Components, Progressive Web Apps, dan optimasi performa.

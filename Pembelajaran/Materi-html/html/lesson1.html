Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the foundation of all websites. Think of it as the skeleton that gives structure to web content. It's not a programming language, but rather a markup language used to:

  • Organize web page content
  • Define headings, paragraphs, and other text elements
  • Embed images, videos, and other media
  • Create links between pages
  • Structure forms for user input

HTML uses tags (words between angle brackets) to mark up content. These tags tell browsers how to display the content. For example, <h1> tags create main headings, while <p> tags create paragraphs.

Basic HTML Structure 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

Structure Breakdown:

  • <!DOCTYPE html> - Declares HTML5 document type
  • <html> - Root element wrapping all content
  • <head> - Contains meta information and page title
  • <title> - Sets browser tab title (not visible on page)
  • <body> - Contains visible page content

Best Practices:

  • Always include the DOCTYPE declaration
  • Use proper indentation (2 or 4 spaces)
  • Close all tags properly
  • Use lowercase for tag names
  • Add comments for complex sections

Key Concepts

1. Elements and Tags

HTML elements consist of:

<tagname>Content goes here...</tagname>

Example: <p>This is a paragraph</p>

2. Self-closing Tags

Some elements don't need closing tags:

<img src="image.jpg" alt="Description">
<br>
<input type="text">

Getting Started

  1. Create a new file with .html extension
  2. Start with the basic HTML structure
  3. Add content between <body> tags
  4. Save and open in web browser

Tip: Use a text editor like VS Code or Sublime Text for better coding experience!

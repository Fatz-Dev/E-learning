Pengenalan HTML

HTML merupakan singkatan dari HyperText Markup Language, yaitu script untuk menyusun dokumen-dokumen Web. Dokumen HTML disimpan dalam format teks reguler dan mengandung tag-tag yang memerintahkan web browser untuk mengeksekusi perintah-perintah yang dispesifikasikan.

1. Penamaan Dokumen HTML

Penamaan dokumen dapat dilakukan pada saat melakukan penyimpanan. Pada saat mengetikkan nama file, maka file diberikan ekstensi .html, contoh: namafile.html.

2. Struktur Dokumen HTML

Dokumen HTML mempunyai tiga buah tag utama yang membentuk struktur dari dokumen HTML, yaitu <html>, <head>, dan <body>. Tag <html> berfungsi untuk menyatakan suatu dokumen HTML, tag <head> berfungsi untuk memberikan informasi tentang dokumen HTML, dan tag <body> berfungsi untuk menyimpan informasi atau data yang akan ditampilkan dalam dokumen HTML. Elemen yang terdapat pada sebuah dokumen HTML terbagi atas dua bagian, yaitu Head dan Body.

a. Bagian Head

Bagian header dari dokumen HTML diapit dengan tag <head>....</head>. Di dalam bagian ini biasanya dimuat tag <title>....</title> yang menampilkan judul dari halaman pada browser.

b. Bagian Body

Dokumen body diapit dengan tag <body>....</body>. Bagian body digunakan untuk menampilkan teks, gambar, link, dan semua elemen yang akan ditampilkan pada halaman web.

Struktur Dasar HTML:

<!doctype html>
        <html>
        <head>
            <title>Judul Dokumen HTML</title>
        </head>
        <body>
            <h1>Heading Pertama</h1>
            <p>Ini adalah paragraf pertama.</p>
        </body>
        </html>

Penjelasan Tag HTML

<!doctype html>

Adalah tag awal dari setiap dokumen HTML. Tag ini berfungsi untuk menginformasikan pada browser bahwa dokumen yang sedang dibuka adalah dokumen HTML.

<html>......</html>

Tag ini menandai awal dari isi dokumen HTML. Semua kode HTML yang dibuat akan ditulis di dalam tag ini.

<head>......</head>

Tag <head> digunakan untuk menyimpan informasi tentang dokumen HTML. Informasi yang disimpan di sini tidak langsung ditampilkan di halaman web, namun penting untuk pengaturan dokumen.

<title>.......</title>

Salah satu contoh informasi yang terdapat di dalam tag <head> adalah <title>. Tag <title>......</title> ini akan menampilkan teks pada judul browser atau pada tab browser.

<body>......</body>

Tag <body> merupakan tag pembuka dari badan dokumen HTML. Semua konten yang akan ditampilkan di halaman web, seperti teks, gambar, dan link, ditulis di dalam tag ini.

3. Elemen dan Tag Pada Dokumen HTML

Ciri utama dokumen HTML adalah adanya tag dan elemen. Elemen dalam dokumen HTML dikategorikan menjadi dua, yaitu elemen <head> yang berfungsi memberikan informasi tentang dokumen tersebut dan elemen <body> yang menentukan bagaimana isi suatu dokumen ditampilkan oleh browser, seperti paragraf, daftar (list), tabel, dan lain-lain.

Sedangkan tag dinyatakan dengan tanda lebih kecil "<" (tag awal) dan tanda lebih besar ">" (tag akhir). Dalam penggunaannya, sebagian besar kode HTML tersebut harus terletak di antara tag kontainer. Yaitu diawali dengan <namatag> dan diakhiri dengan </namatag> (terdapat tanda garis miring "/").

Bagian Head

Bagian <head> digunakan untuk menyimpan informasi tentang dokumen HTML. Elemen-elemen dalam bagian <head> tidak langsung ditampilkan pada halaman web tetapi memberikan informasi penting, seperti judul halaman, metadata, dan link ke sumber daya lain (misalnya CSS).

Bagian Body

Bagian <body> adalah tempat semua konten yang akan ditampilkan pada halaman web berada, seperti teks, gambar, video, dan elemen lainnya. Semua elemen yang terlihat oleh pengguna web akan berada di dalam tag <body>.

Tag dan Elemen dalam HTML

Dokumen HTML terdiri dari elemen-elemen yang saling berkaitan untuk membentuk suatu dokumen HTML yang utuh, antara lain elemen HTML, elemen <head>, dan elemen <body>. Elemen-elemen ini masing-masing dapat diapit tanda yang disebut delimeter (<....>).

Elemen yang telah diapit dengan tanda delimeter disebut dengan tag. Berikut adalah contoh penulisan elemen dalam HTML:

<nama elemen>

Misalnya: <html>

Perlu diingat juga bahwa pada setiap elemen atau tag harus selalu diakhiri atau ditutup dengan menuliskan kembali tag dan menambahkan tanda garis miring "/". Berikut adalah cara penulisannya:

</nama elemen>

Misalnya: </html>

Tag HTML Dasar

Tag Deskripsi
<html> Mendefinisikan sebuah dokumen HTML.
<body> Mendefinisikan bagian body dokumen HTML.
<h1> </h6> Mendefinisikan header 1 sampai 6.
<p> Mendefinisikan paragraf.
<br> Menambahkan line break (spasi baru).
<hr> Membuat garis datar (horizontal).
<!-- --> Mendefinisikan komentar.

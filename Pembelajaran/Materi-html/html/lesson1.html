Pengenalan HTML

HTML merupakan singkatan dari HyperText Markup Language, yaitu script untuk menyusun dokumen-dokumen Web. Dokumen HTML disimpan dalam format teks reguler dan mengandung tag-tag yang memerintahkan web browser untuk mengeksekusi perintah-perintah yang dispesifikasikan.

1. Penamaan Dokumen HTML

Penamaan dokumen dapat dilakukan pada saat melakukan penyimpanan. Pada saat mengetikkan nama file, maka file diberikan ekstensi .html, contoh: namafile.html.

2. Struktur Dokumen HTML

Dokumen HTML mempunyai tiga buah tag utama yang membentuk struktur dari dokumen HTML, yaitu <html>, <head>, dan <body>. Tag <html> berfungsi untuk menyatakan suatu dokumen HTML, tag <head> berfungsi untuk memberikan informasi tentang dokumen HTML, dan tag <body> berfungsi untuk menyimpan informasi atau data yang akan ditampilkan dalam dokumen HTML. Elemen yang terdapat pada sebuah dokumen HTML terbagi atas dua bagian, yaitu Head dan Body.

a. Bagian Head

Bagian header dari dokumen HTML diapit dengan tag <head>....</head>. Di dalam bagian ini biasanya dimuat tag <title>....</title> yang menampilkan judul dari halaman pada browser.

b. Bagian Body

Dokumen body diapit dengan tag <body>....</body>. Bagian body digunakan untuk menampilkan teks, gambar, link, dan semua elemen yang akan ditampilkan pada halaman web.

Struktur Dasar HTML:

<!doctype html>
        <html>
        <head>
            <title>Judul Dokumen HTML</title>
        </head>
        <body>
            <h1>Heading Pertama</h1>
            <p>Ini adalah paragraf pertama.</p>
        </body>
        </html>

Penjelasan Tag HTML

<!doctype html>

Adalah tag awal dari setiap dokumen HTML. Tag ini berfungsi untuk menginformasikan pada browser bahwa dokumen yang sedang dibuka adalah dokumen HTML.

<html>......</html>

Tag ini menandai awal dari isi dokumen HTML. Semua kode HTML yang dibuat akan ditulis di dalam tag ini.

<head>......</head>

Tag <head> digunakan untuk menyimpan informasi tentang dokumen HTML. Informasi yang disimpan di sini tidak langsung ditampilkan di halaman web, namun penting untuk pengaturan dokumen.

<title>.......</title>

Salah satu contoh informasi yang terdapat di dalam tag <head> adalah <title>. Tag <title>......</title> ini akan menampilkan teks pada judul browser atau pada tab browser.

<body>......</body>

Tag <body> merupakan tag pembuka dari badan dokumen HTML. Semua konten yang akan ditampilkan di halaman web, seperti teks, gambar, dan link, ditulis di dalam tag ini.

3. Elemen dan Tag Pada Dokumen HTML

Ciri utama dokumen HTML adalah adanya tag dan elemen. Elemen dalam dokumen HTML dikategorikan menjadi dua, yaitu elemen <head> yang berfungsi memberikan informasi tentang dokumen tersebut dan elemen <body> yang menentukan bagaimana isi suatu dokumen ditampilkan oleh browser, seperti paragraf, daftar (list), tabel, dan lain-lain.

Sedangkan tag dinyatakan dengan tanda lebih kecil "<" (tag awal) dan tanda lebih besar ">" (tag akhir). Dalam penggunaannya, sebagian besar kode HTML tersebut harus terletak di antara tag kontainer. Yaitu diawali dengan <namatag> dan diakhiri dengan </namatag> (terdapat tanda garis miring "/").

Bagian Head

Bagian <head> digunakan untuk menyimpan informasi tentang dokumen HTML. Elemen-elemen dalam bagian <head> tidak langsung ditampilkan pada halaman web tetapi memberikan informasi penting, seperti judul halaman, metadata, dan link ke sumber daya lain (misalnya CSS).

Bagian Body

Bagian <body> adalah tempat semua konten yang akan ditampilkan pada halaman web berada, seperti teks, gambar, video, dan elemen lainnya. Semua elemen yang terlihat oleh pengguna web akan berada di dalam tag <body>.

Tag dan Elemen dalam HTML

Dokumen HTML terdiri dari elemen-elemen yang saling berkaitan untuk membentuk suatu dokumen HTML yang utuh, antara lain elemen HTML, elemen <head>, dan elemen <body>. Elemen-elemen ini masing-masing dapat diapit tanda yang disebut delimeter (<....>).

Elemen yang telah diapit dengan tanda delimeter disebut dengan tag. Berikut adalah contoh penulisan elemen dalam HTML:

<nama elemen>

Misalnya: <html>

Perlu diingat juga bahwa pada setiap elemen atau tag harus selalu diakhiri atau ditutup dengan menuliskan kembali tag dan menambahkan tanda garis miring "/". Berikut adalah cara penulisannya:

</nama elemen>

Misalnya: </html>

Tag HTML Dasar

Tag Deskripsi
<html> Mendefinisikan sebuah dokumen HTML.
<body> Mendefinisikan bagian body dokumen HTML.
<h1> </h6> Mendefinisikan header 1 sampai 6.
<p> Mendefinisikan paragraf.
<br> Menambahkan line break (spasi baru).
<hr> Membuat garis datar (horizontal).
<!-- --> Mendefinisikan komentar.

Heading dan Paragraph

Heading digunakan untuk memberikan penjudulan pada suatu dokumen HTML. Untuk melakukan format heading maka menggunakan tag <h1>....</h1> untuk judul utama dan untuk judul sub bab dapat menggunakan tag <h2>....</h2> sampai dengan <h6>....</h6>. Untuk membuat sebuah paragraph dalam dokumen HTML maka digunakan tag <p>...</p>.

Link

Modul Pemrograman Web Link atau biasa disebut dengan anchor (pengait) dapat dibuat dengan menambahkan tag <a>....</a> serta atribut href agar link tersebut dapat menyimpan alamat web yang akan dituju ketika link diklik. Sehingga sintaks link adalah : 

<a href=’alamat web yang dituju’>...</a>

List

Dalam dokumen HTML suatu daftar atau list adalah suatu kumpulan item tertentu, daftar item dapat ditulis secara berurutan atau tidak. Jenis list dalam HTML adalah:

  • Ordered list: Ordered list dapat dibuat dengan menggunakan tag <ol>....</ol> dan untuk setiap listnya menggunakan tag <li>....</li>.
  • Unordered list: Unordered list dapat dibuat dengan menggunakan tag <ul>....</ul> dan untuk setiap listnya menggunakan tag <li>....</li>.

Image

Dalam dokumen HTML kita dapat memuat dan menampilkan gambar. Untuk memuat gambar menggunakan tag <img> untuk menentukan file gambar maka digunakan atribut src. Sehingga sintaks image adalah :

<img src=’nama file gambar.jpg’>

Form

Sebuah form dalam dokumen HTML dapat memiliki bermacam kontrol, mulai dari text input, combo box, button, dan lain sebagainya. Pembuatan sebuah form diawali dengan menggunakan tag :

<form>....</form>

Label

Label digunakan untuk memberikan keterangan pada setiap input yang ada. Sintaks form label adalah :

<label for=’nama’>keterangan input</label>

Text

Kontrol input ini dapat diisi dengan teks yang memiliki panjang kata terbatas, biasa digunakan dalam form pencarian, nama, dan lainnya. Sintaks form text adalah :

<input type=’text’ name=’nama’>

Combobox

Combobox adalah kontrol yang memiliki pilihan ketika diklik. Sintaks form combobox adalah :

<select>
        <option value=”name value 1”>Name 1</option>
        <option value=”name value 2”>Name 2</option>
        <option value=”name value 3”>Name 3</option>
 </select>

Contoh Form (Combobox)

Tabel

Untuk menampilkan data dengan tipe tabel dalam HTML, dapat menggunakan tag:

<table>....</table>

Tabel

Untuk menampilkan data dengan tipe tabel dalam HTML, dapat menggunakan tag:

          <thead>
            <tr>
              <th>Nama</th>

Penggunaan Tag <div> dalam HTML

Penggunaan tag <div> adalah sebagai container atau pembatas satu struktur HTML dengan struktur lainnya. Dalam struktur HTML terdapat beberapa bagian, antara lain:

a. Header

Bagian kepala website yang berisi logo, nama, slogan, menu, dan lainnya. Sintaks <div> untuk header adalah:

<div id='header'>......</div>

b. Content

Berisi isi website itu sendiri, contoh content berisi postingan terbaru atau informasi lainnya. Sintaks <div> untuk content adalah:

<div id='content'>......</div>

c. Sidebar

Berada di samping content dan biasanya berisi iklan, kategori artikel, widget, dan lainnya. Sintaks <div> untuk sidebar adalah:

<div id='sidebar'>......</div>

d. Footer

Bagian kaki dari website yang pada umumnya diisi dengan tag website tersebut, misalnya copyright ©2018 by me. Sintaks <div> untuk footer adalah:

<div id='footer'>......</div>

KEGIATAN PRAKTIKUM

1. Membuat Heading dan Paragraph

Langkah-langkah praktikum

  • Buka direktori C > xampp > htdocs, kemudian buat folder dan beri nama “praktikum_web”.
  • Buka editor Notepad++ atau Sublime Text.
  • Modul Pemrograman Web.
  • Buatlah file baru dengan membuka menu file > new atau dengan shortcut Ctrl + N.
  • Tuliskan coding berikut:
<!DOCTYPE html>
<html lang="id">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Praktikum Web</title>
    </head>
    <body>
        <h1>Selamat Datang di Praktikum Web</h1>
        <p>Ini adalah paragraf pertama yang saya buat dalam praktikum web. Saya sedang belajar membuat halaman web menggunakan HTML. Semoga praktikum ini dapat membantu saya memahami dasar-dasar HTML.</p>
    </body>
</html>
  • Simpan kode yang telah dibuat dengan memilih menu File > Save As... atau shortcut Ctrl + S
  • Pilih lokasi penyimpanan di folder C > xampp > htdocs > praktikum_web, beri nama file dengan prak1.html, dan pilih Hypertext Markup Language File ( html ) pada Save as type, kemudian klik Save.
  • Buka aplikasi XAMPP, lalu aktifkan Apache dan MySQL dengan menekan tombol Start pada modul Apache dan MySQL.
Buka browser dan ketikkan URL berikut untuk melihat hasil program:

http://localhost/praktikum_web/prak1.html

2. Praktikum Membuat List

Salin koding dibawah ini untuk membuat list menggunakan HTML: 

            
<!doctype html>
<html>
<head>
<title>List</title>
</head>
<body>
<h3>Membuat Unordered List</h3>
<strong><i><u>Bahan-bahan Memasak Telur Goreng</u></i></strong>
<ul>
    <li>Telur 3 butir</li>
    <li>Margarin (1 sdm)</li>
    <li>Batang Daun bawang</li>
    <li>Bawang merah (2 butir)</li>
    <li>Cabai merah keriting (2 buah)</li>
    <li>Lada 1/4 sdt</li>
    <li>Garam 1/4 sdt</li>
</ul>
<h3>Membuat Ordered List</h3>
<strong><i><u>Cara Memasak Telor Goreng</u></i></strong>
<ol>
    <li>Campur semua bahan di atas</li>
    <li>Kocok semua bahan hingga merata</li>
    <li>Panaskan margarin</li>
    <li>Goreng dan dadar hingga kuning keemasan (matang)</li>
    <li>Selesai dan masakan dihidangkan</li>
</ol>
</body>
</html>
  • Simpan kode yang telah dibuat dengan memilih menu File > Save As... atau shortcut Ctrl + S
  • Pilih lokasi penyimpanan di folder C > xampp > htdocs > praktikum_web, beri nama file dengan prak2.html, dan pilih Hypertext Markup Language File ( html ) pada Save as type, kemudian klik Save.
  • Buka aplikasi XAMPP, lalu aktifkan Apache dan MySQL dengan menekan tombol Start pada modul Apache dan MySQL.


Buka browser dan ketikkan URL berikut untuk melihat hasil program: http://localhost/praktikum_web/prak2.html

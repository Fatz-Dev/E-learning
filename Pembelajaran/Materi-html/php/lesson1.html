Introduction to PHP

What is PHP?

PHP (Hypertext Preprocessor) is a server-side scripting language designed specifically for web development. It can be embedded directly into HTML code.

Basic PHP Syntax:

<?php
// Variables
$name = "John";
$age = 25;

// Basic Function
function sayHello($name) {
    return "Hello, " . $name . "!";
}

// Array
$colors = array("red", "green", "blue");

// Associative Array
$person = array(
    "name" => "John",
    "age" => 25,
    "city" => "New York"
);
?>

Key Features

  • Server-side scripting
  • Database integration
  • Form handling
  • Session management
